# Windows BSOD + Win98 风格视觉重设计

> 将 Bug 人格小程序从黑底绿字终端风改为 BSOD 蓝底 + Win98 经典控件的混合风格。

## 配色体系

| 变量名 | 值 | 用途 |
|--------|------|------|
| `--bsod-blue` | `#0000AA` | 全屏背景 |
| `--win-gray` | `#C0C0C0` | 窗口内部底色 |
| `--win-title-blue` | `#000080` | 窗口标题栏 |
| `--win-white` | `#FFFFFF` | 蓝底文字 / 按钮高光 |
| `--win-black` | `#000000` | 窗口内文字 |
| `--win-shadow` | `#808080` | 按钮 3D 阴影 |
| `--win-dark-shadow` | `#404040` | 凹陷区域 |
| `--win-highlight` | `#DFDFDF` | 按钮面亮色 |

## 窗口控件

- 按钮：2px 凸起边框（上/左 #FFFFFF，下/右 #808080），按下时凹陷
- 标题栏：深蓝底 + 白色标题 + `[_][□][X]` 装饰（不可交互）
- 线框：Win98 `GroupBox` 风格，灰色蚀刻线
- 进度条：蓝色方块在白色底上前进
- 单选项：○/● 符号 + 整行可点击

## 三页设计

### 首页
- BSOD 蓝底全屏，中央 Win98 对话框
- 命令行调用模拟 `C:\Users\You> bmti.exe`
- 信息框（灰色线框）展示 7 题/16 型信息
- `[开始诊断 (S)]` 立体按钮
- 底部 BSOD 风格暗文

### 测试页
- 进度条：蓝色块 + 百分比 + `扫描扇区` 文字
- 每题一个 `GroupBox` 线框，标题 `[Q1]` 格式
- 选项 ○ 单选框 + 整行点击
- `[下一步]` 按钮（需点击确认，不自动跳转）

### 结果页
- 报告卡片嵌入窗口，`>` 提示符分组
- 型号名 + 色号（小色块）
- Diagnostic Code + 时间戳
- `[发送给懂你Bug的人 (Alt+S)]` 主按钮
- `[ 重新扫描 (F5) ]` 文字链接
